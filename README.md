# Siddhesh Kosalge — Personal Portfolio Website
 
A responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript — showcasing my projects, skills, certifications, and ways to get in touch.
 
🔗 **Live Site:** [personalportfoliowebsite-seven-cyan.vercel.app](https://personalportfoliowebsite-seven-cyan.vercel.app/)
 
## ✨ Features
 
- 🌙 Dark / light mode toggle
- 📊 Skills section with card and radar-chart views
- 🏆 Certifications with links to view/verify
- 💼 Projects section with filterable tags and detailed case studies (Problem → Solution → Result)
- 📈 Live GitHub stats (repos, followers, following, gists)
- 💬 Recommendations carousel + a form for visitors to leave their own
- 📩 Contact section with a pre-filled email form, WhatsApp chat link, and "Book a Call" option
- 🖨 Print-friendly view
- 📄 Downloadable resume (PDF)
- 📱 Fully responsive, single-page layout
## 🛠 Built With
 
- **HTML5** — semantic structure
- **CSS3** — custom properties (variables), Flexbox-based responsive layouts
- **JavaScript (Vanilla)** — dark mode toggle, project filtering, recommendation form, GitHub API integration, dynamic content rendering
No frameworks, no build tools, no dependencies — just plain HTML, CSS, and JS.

## 🚀 Getting Started
 
To run this project locally:
 
```bash
# Clone the repository
git clone https://github.com/siddheshkosalge896-afk/Personalportfoliowebsite.git
 
# Navigate into the project folder
cd Personalportfoliowebsite
 
# Open index.html in your browser
```
 
No build steps or installations required — it's a static site.
 
## 🌐 Deployment
 
This site is deployed on [Vercel](https://vercel.com/). Any push to the `main` branch automatically redeploys the live site.
